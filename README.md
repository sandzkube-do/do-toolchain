# do-toolchain
DevOps Team Toolchain

A set of tools that enables the DevOps culture of "Automating Everything", "Infrastructure as Code", "Pipeline as code", Workspace as code, and "Relentless Automation".

These tools includes the Kubernetes CLI (kubectl), Openshift CLI (oc), Openshift DO CLI (odo), Private Git Repository (gitea), Openshift Code Ready Containers (OCP4), VS Code, Eclipse Che, Tekton, Nexus, Sonarqube, etc.
